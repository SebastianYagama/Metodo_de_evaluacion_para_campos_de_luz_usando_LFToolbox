En esta carpeta se guarda la implementación creada para manejar campos de luz usando 
cámaras Lytro y se emplean dos carpetas externas para la lectura de dichos campos de luz:
La carpeta “Cameras” contiene las imágenes en blanco y la calibración de las cámaras.
La carpeta “Images” contiene los distintos campos de luz tomadas por las cámaras Lytro usadas.
